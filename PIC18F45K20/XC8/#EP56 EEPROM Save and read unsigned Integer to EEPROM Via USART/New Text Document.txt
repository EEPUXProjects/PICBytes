Welcome to this revamped mini series called PICbytes,

Here you will find various microcontroller tutorials using the PIC18F45K20 Microcontroller. 

In this video we find out how to store a unsigned integer into the EEPROM. 
It is not as simple as a storing a 8 bit char which takes up one address. 
We have to add some extra code to split the value into two 8 bit address's as it is a 16 bit unsigned integer.
Then when we read it back on the serial terminal we need a special function to organise it back into a single 16 bit unsigned integer.
